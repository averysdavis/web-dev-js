# Styling Elements Using JavaScript
`element.setAttribute(name,value);`
use for elements you add with js  



`footNote.setAttribute("style", "font-size:7px;");`
basically css inside the parentheses  



css:
`.footNoteCSS{
font-size:7px;
}`

js:
`footNote.setAttribute("class", "footNoteCSS");`  


attributes are things contained in the opening tag of an element  



`element.attribute.value = "value";
footNote.style.fontSize = "7px";
link.href = "link.com";`
